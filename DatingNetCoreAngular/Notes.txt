view dotnet CLI version 
PS> dotnet --info

add dev cert as trusted
PS> dotnet dev-certs https --trust

run Project
PS> dotnet run

watch files while running
PS> dotnet watch run

install ef tools from nuget - will not work in visul studio
PS> dotnet tool install --global dotnet-ef --version 5.0.11


create initial migration
PS> dotnet ef migrations add InitialCreate -o Data/Migrations

remove migration
PS> dotnet ef migrations remove

run mirgation and update database
PS> dotnet ef database update
